# Food & Beverage Analysis Dashboard (Power BI)

This project contains a Power BI dashboard built to analyze sales, ratings, and product performance in the food & beverage domain.

## Features
- Data cleaning and transformation using Power Query  
- Interactive visuals and KPIs  
- Insights on product performance and customer trends  
- Filters and slicers for easy exploration  

## Tools Used
- Power BI  
- Excel  

## How to Use
1. Download the `.pbix` file
2. Open it in Power BI Desktop
3. Explore the visuals and insights
